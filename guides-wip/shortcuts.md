# Shortcuts



| Tools | Canvas |
| :--- | :--- |
|  `F`  Frame |  `⌘`  Select |
|  `S`  Stack |  `Z`  Zoom |
|  `T`  Text  |  `⌘`   `+`  Zoom In |
|  `L`  Link |  `⌘`   `-`  Zoom Out  |
|  `P`  Path |  `⌘`   `O`  Reset Zoom |
|  `R`  Rectangle |  `⌘`   `1`   Center Canvas |
|  `O`  Oval |  `⌘`   `9`   Focus Canvas |
|  | `⌘` `Drag` Pan |
