# Shortcuts



| Tools | Canvas |
| :--- | :--- |
|  `F`  Frame |  `⌘`  Select |
|  `S`  Stack |  `⌘`   `+`  Zoom In |
|  `T`  Text  |  `⌘`   `-`  Zoom Out |
|  `L`  Link |  `⌘`   `0` Actual Size |
|  `P`  Path |  `⌘`   `1`  Center Canvas |
|  `R`  Rectangle |  `Space`    `Drag`  Pan Canvas |
|  |  |
|  |  |
