# Shortcuts

| Tools | Canvas |
| :--- | :--- |
| `F`  Frame | `⌘`  Select |
| `S`  Stack | `Z`  Zoom |
| `T`  Text | `⌘`   `+`  Zoom In |
| `L`  Link | `⌘`   `-`  Zoom Out |
| `P`  Path | `⌘`   `O`  Reset Zoom |
| `R`  Rectangle | `⌘`   `1`   Center Canvas |
| `O`  Oval | `⌘`   `9`   Focus Canvas |
|  | `⌘` `Drag` Pan |

| Text |
| :--- |
| `⌘`  `T`  Change typeface |
| `⌘`   `↑`  Decrease text size |
| `⌘`   `↓`   Increase text size |
| `Select Text`   `⌘`   `U`   Underline text |
| `Select Text`   `⌘`   `I`   Italic text |
| `Select Text`   `⌘`   `B`   Bold text |
